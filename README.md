# 📊 Dashboard de Ventas - Power BI

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de un dashboard interactivo en **Power BI** para el análisis de ventas, clientes, productos y rentabilidad. El objetivo es facilitar la toma de decisiones mediante indicadores clave (KPIs), visualizaciones dinámicas y filtros interactivos.

El reporte permite explorar el comportamiento de las ventas desde diferentes perspectivas, identificando tendencias, oportunidades de mejora y segmentos con mayor desempeño.

---

## 🎯 Objetivos

- Analizar el comportamiento de las ventas.
- Identificar las regiones y sucursales con mejor desempeño.
- Conocer el perfil de los clientes.
- Evaluar el rendimiento de las categorías de productos.
- Medir la rentabilidad del negocio mediante indicadores clave.
- Apoyar la toma de decisiones basada en datos.

---

## 📑 Contenido del Dashboard

### 🏠 Menú de Inicio
Pantalla principal que facilita la navegación entre las diferentes páginas del reporte mediante botones interactivos.

### 📖 Glosario y Diccionario de Datos
Describe el significado de cada campo utilizado en el modelo de datos y las métricas empleadas en el dashboard.

### 📈 Vista General de Ventas
Presenta una visión global del negocio mediante indicadores como:

- Ventas Totales
- Cantidad Vendida
- Precio Promedio
- Número de Clientes
- Evolución de las ventas
- Participación por categoría y canal

### Análisis por Región o Sucursal
Permite analizar el desempeño comercial según la ubicación geográfica.

Incluye análisis por:

- Región
- Sucursal
- Ventas
- Participación porcentual

###  Análisis de Clientes
Analiza las características de los clientes para identificar segmentos relevantes.

Incluye:

- Distribución por género
- Distribución por edad
- Clientes con mayores compras
- Participación por canal de venta

###  Análisis de Productos
Evalúa el comportamiento de los productos y categorías.

Se analiza:

- Ventas por categoría
- Productos con mayores ingresos
- Cantidad vendida
- Precio promedio

###  Indicadores de Rentabilidad
Presenta los principales indicadores financieros para evaluar el desempeño del negocio.

Ejemplos:

- Ventas Totales
- Ticket Promedio
- Precio Promedio
- Cantidad Vendida
- Rentabilidad por categoría
- Rentabilidad por región

### 📌 Conclusiones y Recomendaciones
Resume los principales hallazgos obtenidos del análisis y propone recomendaciones orientadas a mejorar el desempeño comercial.

# 📂 Diccionario de Datos

| Campo | Descripción |
|---------|------------|
| Num | Identificador único del registro |
| Fecha | Fecha de la venta |
| Región | Región o sucursal donde se realizó la venta |
| Categoría | Categoría del producto vendido |
| Canal | Canal de venta (Tienda, Online, etc.) |
| ID Cliente | Identificador único del cliente |
| Edad_Clie | Edad del cliente |
| Género | Género del cliente |
| Precio_Unita | Precio unitario del producto |
| Cantidad | Cantidad de unidades vendidas |
| Total venta | Importe total de la venta |
